# BawrSpam Changelog

## v1.1.2 (2026-05-26)

- ConfigPanel modernized to native Blizzard UI: sliders, checkboxes, and import / export dialogs now use Blizzard's own widgets. AceGUI is no longer required.
- HistoryPanel size is now fixed at 940 x 560 with a locked list / detail split, so every section renders predictably regardless of monitor size.
- Fix: opening BawrSpam Config from Blizzard's Settings panel no longer prints a UI-blocked warning.
- Fix: switching to the Config tab no longer leaves the list / detail divider drawn across the configuration area.
- Fix: lifetime stat rows in the History detail pane render correctly without the trailing entry clipping into the divider.

## v1.1.1 (2026-05-21)

- Multi-client support: now runs on Retail, Classic Era, and Burning Crusade Classic.
- New repeat-message (flood) detection.
- Expanded and tuned spam pattern coverage.
- Performance and stability improvements.
